The ComparisonTool is a python-based tool for comparing the contents of directories, in particular designed for results of workflows. 

To use the tool, run pip install -e in the directory. 

Then the tool can be used: 

compare directory /path/to/run1/ /path/to/run2/ -c path/to/yourconfig.yaml 

Further information for the config and flags can be found in the usage.md page.
